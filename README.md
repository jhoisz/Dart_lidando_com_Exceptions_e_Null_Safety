## Diferença entre Erro e Exceção

Exceções, diferente de erros, são situações que você pode planejar e resolver depois.

> Exemplos de **exceções**:

- `FormatException`;
- `TimeOutException`;
- `IOException`;
- Exceções personalizadas.

> Exemplos de **erros**:

- `StackOverFlowError`;
- `OutOfMemoryError`;
- `ArgumentError`;
- `TypeError`.
